# 照片备份工具
经常有很多照片通过微信等应用保存到手机，这种照片的文件名会丢失时间信息，通常我们要等很久才会备份照片到电脑，按照时间归类就成了大问题。本软件会扫描同级子目录，自动读取每个文件的创建时间，根据创建时间查询并建立目录，然后将该文件拷贝到目录中去，完成自动归类，实为居家必备神器。本神器支持jpg、png、jpeg、mp4、heic和mov等格式的照片和视频文件。
